# PRD — 영어 사전 (MyDict)

## 1. 제품 개요

Claude AI를 기반으로 한 영어 단어 / 상용구 검색 웹 서비스.  
기존 사전 앱과 달리 AI가 문맥에 맞는 한국어 뜻·예문·어원을 즉시 생성하여 제공한다.

- **서비스 URL:** https://mydict-production.up.railway.app
- **현재 버전:** 1.0.0
- **배포 환경:** Railway (Node.js)

---

## 2. 목표

| 목표 | 지표 |
|------|------|
| 영어 단어/상용구를 한국어로 빠르게 이해 | 검색 → 결과 표시까지 5초 이내 |
| 개인 전용 서비스 운영 | 비밀번호 인증으로 무단 접근 차단 |
| 유지비 최소화 | Claude Haiku 모델 사용 (저비용) |

---

## 3. 사용자

- **1차 사용자:** 서비스 소유자 (개인 사용)
- **확장 가능:** 비밀번호를 공유받은 소수 지인

---

## 4. 현재 구현된 기능 (v1.0)

### 4-1. 단어 검색
- 영어 단어 또는 상용구 입력 후 검색
- Enter 키 또는 검색 버튼으로 실행
- 검색 중 로딩 스피너 표시

### 4-2. 결과 카드
| 항목 | 내용 |
|------|------|
| 단어 | 검색어 표시 |
| 발음 | IPA 기호 |
| 품사 | noun / verb / adjective 등 |
| 뜻 | 한국어 의미 (2~4개) |
| 예문 | 영어 예문 + 한국어 번역 |
| 동의어 | 칩(chip) 형태 표시 |
| 반의어 | 칩(chip) 형태 표시 |
| 어원 | 한국어 설명 (선택적) |

### 4-3. 접근 제한
- 첫 방문 시 비밀번호 입력 모달 표시
- 인증 성공 시 `localStorage`에 상태 저장 → 재방문 시 자동 통과
- `APP_PASSWORD` 미설정 시 인증 없이 통과 (개발 편의)

### 4-4. 에러 처리
- 유효하지 않은 영어 단어 입력 시 오류 메시지
- 서버/API 오류 시 사용자 친화적 메시지 표시

## 5. 기술 스택

| 구분 | 기술 |
|------|------|
| Backend | Node.js + Express |
| Frontend | Vanilla HTML / CSS / JS |
| AI | Claude Haiku (`claude-haiku-4-5-20251001`) |
| 배포 | Railway |
| 인증 | 비밀번호 + localStorage |

---

## 6. API 엔드포인트

| 메서드 | 경로 | 설명 |
|--------|------|------|
| POST | `/api/search` | 단어 검색 (Claude 호출) |
| POST | `/api/auth` | 비밀번호 인증 |
| POST | `/api/ai` | 범용 AI 프롬프트 (미사용) |
| GET | `/api/debug` | 환경변수 진단 (운영 중 삭제 권장) |

---

## 7. 환경변수

| 변수명 | 필수 | 설명 |
|--------|------|------|
| `ANTHROPIC_API_KEY` | ✅ | Claude API 키 |
| `APP_PASSWORD` | 선택 | 서비스 접근 비밀번호 |
| `PORT` | 자동 | Railway가 자동 주입 |
