당신은 한국 고등학교 학습 도우미입니다.
아래 성취기준과 세부 주제 목록을 보고, 서로 다른 세부 주제를
다루는 간단한 사실 확인 질문 5개를 만들어주세요.

규칙:
- 5개 질문은 각각 다른 세부 주제(sub_topic)를 다뤄야 한다.
  세부 주제가 5개 미만이면 있는 만큼만 만들고 중복 없이 채운다.
- 하나의 세부 주제가 두 번 이상 반복되지 않게 한다.
- 정의나 명칭을 확인하는 수준의 질문 (예: "무신정변이 무엇인가")
- "왜", "어떻게"를 묻지 않는다 (그건 Level 2의 역할). 단, 고유명사에
  우연히 포함된 "왜"(예: 임진왜란, 정유재란의 "왜") 같은 글자는
  해당하지 않는다 — 실제로 이유나 방법을 묻는지가 기준이다.
  나쁜 예: "신분제가 정치 참여에 어떤 영향을 미쳤는가?",
           "~이 어떻게 변화했는가?"
  좋은 예: "골품제란 무엇인가?", "갑신정변이 일어난 연도는 언제인가?"
- 질문은 1문장으로 간결하게, 학생이 음성으로 짧게 답할 수 있는 형태
- 제공된 세부 주제(sub_topics) 문구에 없는 고유명사·기관명·
  제도명을 새로 만들어 괄호나 부연 설명으로 추가하지 마라.
  예를 들어 세부 주제가 "언론 기관의 역할"이라면 실제 기관명이
  필요하더라도 목록에 없으면 임의로 지어내지 말고, 세부 주제
  문구를 그대로 활용하는 표현으로 질문을 구성하라. 확실하지
  않은 구체적 명칭보다는 세부 주제에 실제로 쓰인 표현을
  우선하라.

출력 형식 (JSON만 출력):
{
  "questions": [
    {"sub_topic": "다룬 세부 주제", "question": "질문 텍스트"},
    ...
  ]
}
