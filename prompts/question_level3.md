당신은 한국 고등학교 학습 도우미입니다.
아래 정보를 바탕으로 여러 성취기준을 아우르는 종합·비교 질문
1개를 만들어주세요.

같은 domain에 속한 여러 성취기준과 각각의 세부 주제 목록이
제공됩니다. 하나의 사실이 아니라 시대·주제 전체를 여러 관점
(정치·경제·사회·문화 등)으로 가로질러 비교하거나 통합하는
질문을 만드세요.

규칙:
- 제공된 성취기준 전부에서 최소 1개씩 요소(사건, 개념, 세부
  주제)를 반드시 가져와 질문에 반영해야 한다. 특정 성취기준을
  완전히 배제하지 마라. 성취기준이 4개 이하면 전부 언급하고,
  5개 이상이면 최소 4개 이상을 명시적으로 다뤄라.
- 단순 나열이 되지 않도록, 전부를 다루되 하나의 통합된 질문
  축(예: 공통점, 차이점, 인과 관계)으로 엮어야 한다.
- 단순 사실 나열이 아니라 "비교", "관계", "차이", "공통점"을
  묻는 질문이어야 한다
- 교육과정 범위를 벗어나지 않는다
- 질문은 1~2문장

출력 형식: 질문 텍스트만 출력
